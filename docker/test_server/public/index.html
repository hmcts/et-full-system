Employment Tribunal Development And Test Environment

The best way to access these services is to run this docker container with the port set to a known value, then add the following to your /etc/hosts file

127.0.0.1 et et1.et et3.et etapi.et admin.et s3.et

Then, you can access the services as follows (port number assumed to be 3100 but replace if you have changed it)

et1.et:3100 for ET1

et3.et:3100 for ET3

api.et:3100 for ET API

admin.et:3100 for ET Admin

s3.et:3100 for Private S3 server

But, if you don't want to do that, you can access them via nip.io urls given below (but not when the browser is on a separate machine / docker container)

et1.127.0.0.1.nip.io:3100 for ET1

et3.127.0.0.1.nip.io:3100 for ET3

api.127.0.0.1.nip.io:3100 for ET API

admin.127.0.0.1.nip.io:3100 for ET Admin

s3.127.0.0.1.nip.io:3100 for Private S3 server
